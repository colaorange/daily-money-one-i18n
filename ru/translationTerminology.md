## Style Rules

- Keep `Daily Money One` and `Daily Money` untranslated.
- Use `–ö–Ϋ–Η–≥–Α` for `Book`.
- Use `–Γ―΅―ë―²` for accounting `Account`.
- Use `–û–Ω–Β―Ä–Α―Ü–Η―è` for `Transaction`.
- Use `–®–Α–±–Μ–Ψ–Ϋ –Ψ–Ω–Β―Ä–Α―Ü–Η–Η` for `Transaction Template`.
- Use `–‰―¹―²–Ψ―΅–Ϋ–Η–Κ` for `Source` and withdrawal-side concepts.
- Use `–ù–Α–Ζ–Ϋ–Α―΅–Β–Ϋ–Η–Β` for `Destination` and deposit-side concepts.
- Use `–Γ―É–Φ–Φ–Α ―¹–Ω–Η―¹–Α–Ϋ–Η―è` for `Amount Out`.
- Use `–Γ―É–Φ–Φ–Α –Ζ–Α―΅–Η―¹–Μ–Β–Ϋ–Η―è` for `Amount In`.
- Use `–ë–Α–Μ–Α–Ϋ―¹–Ψ–≤―΄–Ι –Ψ―²―΅―ë―²` for `Balance Sheet`.
- Use `–ë–Α–Μ–Α–Ϋ―¹` for `Balance`.
- Use short, natural Russian UI labels. Prefer concise nouns for tabs, filters, buttons, and chart labels.
- Translate product-specific accounting terms by Daily Money One meaning, not by generic banking, trading, or login/account wording.
