## Style Rules

- Use `en` as the primary source.
- Keep `Daily Money One` and `Daily Money` untranslated.
- Use `Libro` for `Book`.
- Use `Cuenta` for accounting `Account`.
- Use `cuenta de usuario`, `cuenta de Google`, or `inicio de sesiﾃｳn` only when the source clearly refers to a login account, Google account, user account, authorization account, or credential account.
- Use `Transacciﾃｳn` for `Transaction`.
- Use `Plantilla de transacciﾃｳn` for `Transaction Template`.
- Use `Plantilla` when the UI context is already under transactions.
- Use `Origen` for `Source` or withdrawal-side concepts.
- Use `Destino` for `Destination` or deposit-side concepts.
- Use `Hoja de balance` for `Balance Sheet`.
- Use `Saldo` for the balance concept in shorter UI labels.
- Keep UI labels short and consistent. Prefer `Nueva transacciﾃｳn`, `Editar transacciﾃｳn`, `Transacciones` over longer explanatory wording.
