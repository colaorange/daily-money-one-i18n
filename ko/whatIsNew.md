## 새로운 기능

### 버전 1.11.0
이번 릴리스에는 다음 기능 업데이트가 포함됩니다.
* 버튼 아이콘에 그림자가 있으면 길게 눌렀을 때 관련 추가 기능이 표시된다는 뜻입니다.
* 홈 화면에서 거래 템플릿을 사용할 수 있는 바로가기를 추가하고, 템플릿 일정을 빠르게 다시 조정하는 기능을 강화했습니다.
* 최신 AI 모델을 사용해 모든 언어 번역을 다시 조정했습니다.

### 버전 1.10.0
이번 릴리스에는 다음 기능 업데이트가 포함됩니다.
* 홈 화면에서 잔액표를 더 강조해 월별 지출을 한눈에 명확하게 볼 수 있습니다.
* 반복 예산 설정을 지원해 이번 달 남은 잔액을 더 쉽게 추적할 수 있습니다.

### 버전 1.9.0
사용자 요청과 Google 정책에 따른 기능 업데이트가 포함됩니다.
* Google API 대상 수준을 35로 업그레이드했습니다.
* 기간 목록, 잔액, 차트의 월 시작일과 연 시작일을 설정할 수 있습니다.
* 검색 결과에 잔액 요약이 바로 표시됩니다.
* Google Drive 업로드 기능을 다시 작성했습니다. 파일은 먼저 로컬에 저장되고 네트워크를 사용할 수 있을 때만 업로드됩니다.

### 버전 1.8.0
사용자 요청에 따른 기능 업데이트가 포함됩니다.
* **메모 개선**: 줄바꿈이 있는 여러 줄 메모를 지원하고 입력 필드에 지우기 버튼을 추가했습니다.
* **계정 검색 옵션 개선**: 거래 검색에 출금 계정과 입금 계정 조건을 추가했습니다.
* **반대 초기 금액 허용**: 초기 금액이 음수일 때 반대 계정의 초기 금액 방향을 설정할 수 있습니다.
* **네덜란드어 지원 추가**.

### 버전 1.7.0
사용자 요청에 따른 기능 업데이트가 포함됩니다.
* **스와이프 전 선택 필요**: 활성화하면 스와이프 작업 전에 항목을 먼저 선택해야 합니다. 실수로 스와이프하는 것을 방지하고 렌더링 성능을 크게 향상하지만, 선택 단계가 하나 더 필요합니다. (환경설정 > 기능 설정)
* **메모 기본 펼침**: 활성화하면 거래 목록의 거래 메모가 기본적으로 펼쳐집니다. (환경설정 > 표시 설정 > 거래 목록 설정)
* **거래 배경색 옵션**: 거래 목록에서 거래의 배경색을 변경할 수 있습니다. 기본값(테마 배경색), 출금 계정, 입금 계정을 선택할 수 있습니다. (환경설정 > 표시 설정 > 거래 목록 설정)
* **거래 고급 모드 편집 개선**: 고급 편집 모드에서 모든 이체 항목을 재정렬하고 원하는 순서로 삭제할 수 있습니다. 이전에는 마지막 이체만 삭제할 수 있었고 재정렬은 불가능했습니다.
* **홈 화면 바로가기 설정**: 월말 거래, 월말 잔액표, 월말 잔액 차트 옵션을 추가했습니다.
