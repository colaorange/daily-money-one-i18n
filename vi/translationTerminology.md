## Style Rules

- Use natural Vietnamese wording for mobile app UI.
- Use sentence case for Vietnamese labels and headings; capitalize only proper nouns and product names.
- Keep `Daily Money One`, `Daily Money`, `Daily Money+`, `Google Drive`, `CSV`, `JSON`, and `XLSX` unchanged.
- Use `Sổ` for the app data container; do not translate it as `Sổ cái` or a formal accounting ledger term.
- Use `Tài khoản` for Daily Money One accounting accounts. When the context clearly means login identity, use `tài khoản đăng nhập`, `tài khoản Google`, or another identity-specific phrase.
- Use `Giao dịch` for app transaction records.
- Use `Nguồn` and `Đích` for transaction directions.
- Keep UI labels concise, especially tabs, buttons, filters, and chart labels.
