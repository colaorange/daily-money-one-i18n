## Neuigkeiten

### In der Version 1.11.0
Diese Version enthält Funktionsupdates:
* Ein Schaltflächensymbol mit Schatten entdeckt? Versuchen Sie es gedrückt zu halten, um eine weitere Funktion anzuzeigen!
* Wir haben Verknüpfungen zur Auswahl von Buchungsvorlagen auf dem Startbildschirm hinzugefügt, sodass das Umplanen super einfach und schnell geht!
* Sie können jetzt beim Erstellen eines neuen Buchs andere Konten kopieren! Das Kopieren anderer Konten wird auch auf dem Konten-Bildschirm unterstützt.
* Die neueste KI hat die Übersetzungen in allen Sprachen optimiert.

### In der Version 1.10.0
Diese Version enthält Funktionsupdates:
* Die Bilanz auf dem Startbildschirm wird hervorgehoben, damit Ihre monatlichen Ausgaben auf einen Blick sichtbar sind.
* Wiederkehrende Budgeteinstellungen werden unterstützt, damit Sie Ihr verbleibendes Budget für den aktuellen Monat leichter verfolgen können.

### In der Version 1.9.0
Diese Version enthält Funktionsupdates auf Grundlage von Nutzerwünschen und Google-Richtlinien:
* Google API Target auf Level 35 aktualisiert.
* Sie können nun den ersten Tag des Monats und Jahres für Zeitraumlisten, Salden und Diagramme festlegen.
* Die Saldenübersicht erscheint nun direkt in den Suchergebnissen.
* Die Google Drive-Upload-Funktion wurde neu geschrieben: Dateien werden zuerst lokal gespeichert und nur hochgeladen, wenn Netzwerk verfügbar ist.

### In der Version 1.8.0
Diese Version enthält ebenfalls Funktionsupdates auf Grundlage von Nutzerwünschen:
* **Verbesserte Notizen**: Unterstützt mehrzeilige Notizen mit Zeilenumbrüchen und ergänzt eine Leeren-Schaltfläche im Eingabefeld.
* **Verbesserte Kontosuche**: Ergänzt Bedingungen für Quelle- und Zielkonten in der Buchungssuche.
* **Umgekehrten Anfangsbetrag erlauben**: Wenn Ihr Anfangsbetrag negativ ist, können Sie die Richtung des Anfangsbetrags für das Gegenkonto festlegen.
* **Niederländische Sprachunterstützung hinzugefügt**.

### In der Version 1.7.0
Diese Version enthält Funktionsupdates auf Grundlage von Nutzerwünschen:
* **Auswahl vor dem Wischen erforderlich**: Wenn aktiviert, müssen Elemente ausgewählt werden, bevor Wischaktionen ausgeführt werden können. Das verhindert versehentliches Wischen und verbessert die Renderleistung deutlich, erfordert aber einen zusätzlichen Auswahlschritt. (Einstellungen > Funktionseinstellungen)
* **Notiz standardmäßig geöffnet**: Wenn aktiviert, wird die Buchungsnotiz in der Buchungsliste standardmäßig geöffnet. (Einstellungen > Anzeigeeinstellungen > Buchungslisteneinstellungen)
* **Optionen für Buchungshintergrundfarbe**: Sie können jetzt die Hintergrundfarbe von Buchungen in der Buchungsliste ändern. Verfügbare Optionen sind Standard (Design-Hintergrundfarbe), Quellkonto und Zielkonto. (Einstellungen > Anzeigeeinstellungen > Buchungslisteneinstellungen)
* **Verbesserte Bearbeitung im erweiterten Buchungsmodus**: Im erweiterten Bearbeitungsmodus können Sie jetzt alle Übertragungen neu anordnen und in beliebiger Reihenfolge löschen. Zuvor konnte nur die letzte Übertragung gelöscht werden, und Neuordnung war nicht möglich.
* **Startbildschirm-Verknüpfungen**: Ergänzt wurden folgende Optionen: Monatsendbuchungen, Monatsendbilanz, Monatsenddiagramm.
